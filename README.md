# Retail Sales Analysis Dashboard
## Project Overview
Developed an interactive Excel dashboard to analyze retail sales performance, profitability, order volume, category performance, regional contribution, and salesperson effectiveness. The project transforms raw sales data into meaningful business insights using Pivot Tables, Pivot Charts, and dashboard visualizations.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Analysis
- Dashboard Design

- KPI Monitoring Dashboard

## Key Business Insights

### Regional Performance

- South region contributed the highest share of sales at approximately 30%.
- Sales were distributed across multiple regions, ensuring balanced business performance.

### Category Analysis

- Electronics generated the highest sales revenue among all categories.
- Laptops and Monitors contributed significantly to overall sales performance.

### Salesperson Performance

- Performance comparison was conducted across multiple sales representatives including Arun, Kiran, Priya, Ravi, and Sita.
- The dashboard helps identify top-performing salespeople and sales improvement opportunities.

### Monthly Sales Trends

- Monthly sales performance was tracked from January to August.
- Peak sales periods can be identified to support inventory and marketing planning.

### Payment Method Analysis

- UPI emerged as a major payment method, supporting digital transaction trends.
- Payment analysis helps understand customer purchasing behavior.
